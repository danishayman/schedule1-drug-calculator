Substance Mixing Rules

When a substance is added to a strain, it may define rules that alter the current list of effects on that strain. Each rule checks for a specific effect and replaces it with another.

These rules are applied in order of the substances added. For each substance, all of its replacement rules are evaluated against the existing effects at that moment. If an effect targeted by a rule is present, it is replaced by the specified new effect immediately.

After all rules of a substance are processed, the substance's default effect is added to the mix if the strain has fewer than 8 total effects. If the strain already has 8 effects, the default effect is skipped.

Replacement effects can themselves be replaced by future substances if applicable. This means the order in which substances are mixed directly affects the final result. All effects are capped to a maximum of 8 and duplicates are removed automatically.
